Java 字符集

发表于 2018-07-13 | 分类于 Java

字符集

将字符char 组成一个集合，就是字符集了。

常见的字符集：

1. 英文字符集
2. 中文字符集
3. Unicode 字符集：包括世界上所有的语言字符的字符集

编码

基本数据类型char 是表示字符的类型，它可以和int 类型之间随意互相赋值。
实际上，char 与int 都是数字，但Java 将char 看作是字符。
即，字符在计算机中就是数字，但它会被计算机“映射”为一个字符。
其中，这个“数字”就是这个字符的编码。
编码的集合就是编码集，简称编码。
因为计算机只能处理数字，所以使用编码集是为了让计算机能够处理字符。

编码和操作系统有密切的关系。
Java 内部使用的是UTF-16 编码。

常见的编码集：

1. ASCII 编码：对应英文字符集
2. GBK 编码：对应中文字符集
3. UTF-8：最常用的Unicode 字符集的编码
4. UTF-16：Unicode，与UTF-8 映射关系不同

编码集 + 字符集 -> 字符 + 字体文件 -> 字形
