发表于 2018-07-12 | 分类于 Java

泛型简介

主要功能：让类型也能够作为参数，并在使用时确定参数。

为什么要使用泛型：因为列表中可以放入任意类型的元素，所以，一个元素放入再取出来之后，也就被抹去了原有的类型信息，被当作了Object 类型。想要恢复原有的类型，必须进行强制类型转换。
如果不能确定元素确切的类型，那么强制类型转换就会有一个ClasCastException 异常。
所以，强制类型转换不是一个鼓励使用的功能，我们可以使用泛型来避免。

不使用泛型，则有时必须使用强制类型转换。
尽量不要使用强制类型转换，因为它可能会造成异常。
