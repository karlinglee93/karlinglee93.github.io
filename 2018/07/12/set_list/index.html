Java 集合类框架 - List接口与Set接口

发表于 2018-07-12 | 分类于 Java

简介

集合类框架是对实现了java.util.Collection 接口的类的统称，用于存储一组元素。

常用子接口：List Set Queue

列表List 接口

java.util.List 接口中主要定义了一些存放、获取和删除元素的方法，如同一个列表。

常用方法

void add(Object) 方法：将指定的元素添加到列表的末尾。
void add(int, Object) 方法：将指定的元素插入到列表的指定位置。其他元素依次后移。
Object get(int)：得到列表中指定位置的元素值。
Object remove(int)：删除列表中指定位置的元素值。其他元素依次前移。
void set(int, Object)：使用指定元素，更改列表中指定位置的元素值。
int size()：返回列表当年的大小

上述方法只是List 接口给出的抽象方法，在具体使用时还是要使用实现了这个接口的类。
LrrayList 和LinkedList 实现了List 接口。

ArrayList 类实现List 接口

ArrayList 类在内部通过一个Object 类的数组来实现List 接口中定义的这些方法。

Object 类的引用可以指向任何对象。ArrayList 类中使用的是Object 数组，所以ArrayList 类中的元素可以是任何类型的引用。
同时，ArrayList 类的对象还会在增加一个新元素的时候自动检查数组的大小，并在必要时创建一个更大的数组，将数据复制过去。

集合Set 接口

集合是同一类元素的聚集，且其中的元素不重复，且元素无序排列.
java.util.Set 接口则是以集合的概念为基础，规定了一组对集合进行操作的方法。

常用方法

boolean add(Object) 方法：向集合中增加一个元素。返回值代表是否增加成功。
boolean remove(Object) 方法：从集合中删除某个与参数相等的元素。成功地找到并删除，返回true
boolean contains(Object) 方法：在集合中寻找与参数相等的元素。找到返回true（代替get() ）
Object[] toArray() 方法：返回一个包含集合中所有元素的数组。
int size() 方法： 返回集合中元素的个数。

HashSet 类实现Set 接口

因为Set 中的元素是无序的，所以没有List 接口中类似的 get(int) 方法。
为了遍历Set 中的元素，可以使用toArray() 方法。
