发表于 2018-07-12 | 分类于 Java

认识Map

健值对

和List 不同（只能够通过int 类型的索引得到相应的元素），Map是用来处理健值对的。
而Map 则允许使用任意的对象作为索引，来得到与之对应的对象。
这个索引对象就是所谓的“键”，而与键对应的对象，就是所谓的“值”。
Map 中，键和值都是Object 类型的引用。所以任何对象都可以用做键和值。

一个键只能对应一个值，一个值可以对应多个键。
